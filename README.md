# ZSSN Api - laravel

## Getting Started
Enter on api folder

    composer install
    
Create the *.env* file, use as example the *.env.example* file and create your database

Run migrations:
 
    php artisan migrate

Run seeds to insert basic items:
 
    php artisan db:seed

Run server:
 
    php artisan serve
    
Swagger Documentation on:

    http://localhost:8000/swagger/
